# Full-Stack E-Commerce Web App

A MERN-based e-commerce platform with product search, shopping cart, order management, secure admin dashboard, and JWT-based authentication.

## 🚀 Features

- **Full Software Development Life Cycle (SDLC)**: Managed from planning to deployment.
- **E-Commerce Functionality**:
  - Product search
  - Shopping cart
  - Order management
- **Admin Dashboard**:
  - Inventory control
  - Order tracking
  - Image uploads with Cloudinary
- **Authentication & Security**:
  - JWT-based authentication
  - Role-based access control
  - Protected routes with Redux

## 🛠 Tech Stack

**Frontend**:
- React.js
- Redux Toolkit
- Tailwind CSS

**Backend**:
- Node.js
- Express.js

**Database & Cloud**:
- MongoDB
- Cloudinary
